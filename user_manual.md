# 智能书城管理系统 - 用户使用手册

## 概述

欢迎使用智能书城管理系统！这是一个现代化的在线图书购买平台，集成了AI智能客服功能，为您提供便捷的购书体验。

## 功能特性

- 🔍 **智能搜索**: 支持书名、作者、关键词搜索
- 🤖 **AI客服**: 24小时智能客服在线咨询
- 📚 **商品推荐**: 基于您的浏览和购买历史推荐图书
- 💝 **收藏心愿**: 收藏喜欢的图书，添加到心愿单
- 🛒 **购物车**: 便捷的购物车管理
- 📦 **订单管理**: 完整的订单跟踪流程
- 👤 **个人中心**: 管理个人信息和地址

## 快速开始

### 1. 用户注册与登录

#### 1.1 注册新账户
1. 访问系统首页
2. 点击右上角"注册"按钮
3. 填写注册信息：
   - 用户名（3-20个字符）
   - 密码（6-20个字符）
   - 邮箱地址
   - 手机号码（可选）
4. 点击"立即注册"完成注册

#### 1.2 用户登录
1. 点击右上角"登录"按钮
2. 输入用户名和密码
3. 点击"登录"进入系统

### 2. 浏览和搜索图书

#### 2.1 首页浏览
- **轮播图**: 展示热门推荐图书
- **分类导航**: 按分类浏览图书
- **热门推荐**: 系统推荐的热门图书
- **最新上架**: 最新添加的图书

#### 2.2 搜索功能
1. **关键词搜索**: 在搜索框输入书名、作者或关键词
2. **分类筛选**: 选择特定分类进行筛选
3. **排序选项**:
   - 按价格排序（低到高/高到低）
   - 按评分排序
   - 按时间排序（最新/最旧）
4. **高级筛选**: 价格区间、评分范围等

#### 2.3 商品详情页
- **图书信息**: 封面、标题、作者、价格、评分
- **详细描述**: 图书内容简介
- **用户评价**: 其他用户的评分和评论
- **相关推荐**: 相似图书推荐

### 3. 使用AI智能客服

#### 3.1 启动AI客服
1. 点击页面右下角的蓝色客服图标
2. AI客服窗口自动打开
3. 系统会显示欢迎消息

#### 3.2 与AI客服对话
- **商品咨询**: "推荐一些编程类书籍"
- **价格询问**: "这本书有优惠吗？"
- **库存查询**: "《Java核心技术》还有库存吗？"
- **购买建议**: "适合初学者的Python书籍有哪些？"
- **订单问题**: "我的订单什么时候发货？"

#### 3.3 AI客服功能特点
- **智能理解**: 理解自然语言询问
- **商品推荐**: 根据需求推荐适合的图书
- **实时回复**: 快速响应用户问题
- **多轮对话**: 支持连续对话
- **商品链接**: 直接提供商品详情链接

### 4. 购物流程

#### 4.1 加入购物车
1. 在商品详情页选择数量
2. 点击"加入购物车"按钮
3. 可继续购物或查看购物车

#### 4.2 管理购物车
1. 点击右上角购物车图标
2. 查看已选商品列表
3. 修改商品数量或删除商品
4. 查看总价格

#### 4.3 创建订单
1. 在购物车中点击"去结算"
2. 选择或添加收货地址
3. 确认订单信息和金额
4. 点击"提交订单"

#### 4.4 支付订单
1. 选择支付方式
2. 确认支付金额
3. 完成支付流程
4. 查看订单状态

### 5. 个人中心管理

#### 5.1 个人信息管理
1. 进入"个人中心"
2. 点击"个人信息"标签
3. 修改昵称、邮箱、手机号
4. 更换头像
5. 保存修改

#### 5.2 收货地址管理
1. 进入"地址管理"页面
2. 添加新地址：
   - 收货人姓名
   - 联系电话
   - 详细地址
   - 设置为默认地址（可选）
3. 编辑或删除现有地址

#### 5.3 订单管理
1. 查看订单列表
2. 按状态筛选订单：
   - 待支付
   - 已支付
   - 已发货
   - 已完成
   - 已取消
3. 查看订单详情
4. 取消未支付订单

### 6. 收藏和心愿单

#### 6.1 收藏图书
1. 在商品详情页点击"收藏"按钮
2. 图书添加到收藏列表
3. 在个人中心查看收藏的图书

#### 6.2 心愿单管理
1. 点击商品的"加入心愿单"按钮
2. 在个人中心查看心愿单
3. 从心愿单直接加入购物车

### 7. 评价和反馈

#### 7.1 商品评价
1. 购买并收到商品后
2. 进入订单详情页
3. 点击"评价"按钮
4. 填写评分（1-5星）和评价内容
5. 提交评价

#### 7.2 客服反馈
- 通过AI客服提出建议
- 联系客服邮箱反馈问题
- 参与用户满意度调查

## 常见问题解答

### Q1: 如何找到想要的图书？
A: 您可以通过以下方式查找图书：
- 使用搜索框输入书名或关键词
- 按分类浏览图书
- 查看系统推荐
- 向AI客服咨询推荐

### Q2: AI客服能解决什么问题？
A: AI客服可以帮助您：
- 推荐适合的图书
- 查询商品信息和库存
- 解答购买相关问题
- 提供订单状态信息
- 协助解决技术问题

### Q3: 如何修改或取消订单？
A: 
- 未支付订单：可在个人中心直接取消
- 已支付订单：请联系客服处理
- 订单信息修改：请在提交前仔细确认

### Q4: 支持哪些支付方式？
A: 目前支持：
- 支付宝
- 微信支付
- 银行卡支付

### Q5: 如何查询物流信息？
A: 
1. 进入个人中心的订单管理
2. 点击已发货订单的"查看物流"
3. 查看实时物流信息

### Q6: 收藏和心愿单有什么区别？
A: 
- **收藏**: 保存感兴趣的图书，方便再次查看
- **心愿单**: 暂时不购买但有购买意向的图书列表
